# Pygame Ball Movement

A simple Pygame project where a blue ball moves to the mouse click position.

## Features
- 500x500 Pygame window
- Blue ball drawn on the screen
- Ball moves when the mouse is clicked

## Requirements
- Python 3.x
- pygame

## How to Run
```bash
pip install pygame
python ball_game.py
```
